# 🌐 Language Translation Tool
### CodeAlpha AI Internship — Task 1

A web-based language translation tool built with Python (Flask) and the `deep-translator` library. Supports 20+ languages with a clean, modern UI.

## Features
- Translate text between 20+ languages
- Auto-detect source language
- Swap source and target languages
- Copy translated text with one click
- Ctrl+Enter keyboard shortcut

## Tech Stack
- **Backend:** Python, Flask
- **Translation:** deep-translator (Google Translate)
- **Frontend:** HTML, CSS, JavaScript

## How to Run

1. Clone the repository:
```
git clone https://github.com/YOUR_USERNAME/CodeAlpha_LanguageTranslationTool
cd CodeAlpha_LanguageTranslationTool
```

2. Install dependencies:
```
pip install -r requirements.txt
```

3. Run the app:
```
python app.py
```

4. Open your browser and go to:
```
http://127.0.0.1:5000
```
